01 Debugger · Windows Internals

Damian Debugger

프로세스 메모리와 PE 정보를 직접 확인하며 프로그램의 실행 흐름을 분석하기 위한 디버거 프로젝트입니다. RAM에 올라온 모듈 정보, PE 헤더, 함수 흐름, 디버깅 이벤트를 다루며 Windows 내부 구조를 학습했습니다.

  • 프로세스 메모리, 모듈, PE 구조 확인 흐름 정리
  • 디버깅 이벤트와 분석자가 보는 정보의 관계 학습
  • CLI 기반 디버깅 인터페이스 확장 방향 검토
C/C++ WinAPI PE Debugger
02 CTF · Reverse Engineering

Reversing CTF Problems

동아리 내부 해킹 대회에서 사용할 리버싱 분야 CTF 문제를 제작하고 풀이 흐름을 설계했습니다. 참가자가 디컴파일, 문자열 분석, 조건문 추적, 디버깅 과정을 자연스럽게 따라갈 수 있도록 문제 난이도와 힌트 흐름을 구성했습니다.

  • DecompileMe, DoNotDebug, FindFunction, FindName, WhatIsYourPassword 등 문제 정리
  • 풀이자가 막히는 지점을 고려한 분석 흐름 설계
  • 문제 제작 후 write-up 형태로 풀이 과정 문서화

  • WPA3-SAE 인증 구조와 Kyber 적용 지점 이해
  • liboqs 연동 환경과 네트워크 테스트 과정 정리

04 Android · Runtime Analysis

Android Reverse Engineering Study

  • Xposed 기반 런타임 후킹 구조 학습
  • ProGuard 난독화 코드 리네이밍 자동화 아이디어 구상
